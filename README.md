Небольшое десктопное приложение для работы с моделями через OpenRouter API.  
UI реализован на Flet.

Приложение поддерживает:
- выбор модели
- отправку сообщений
- сохранение истории диалога
- просмотр простой аналитики
- локальное кэширование сообщений

## Авторизация

При первом запуске приложение запрашивает API-ключ OpenRouter.

Если ключ проходит проверку и баланс аккаунта положительный:
- автоматически генерируется 4-значный PIN
- PIN сохраняется в базе в виде хэша
- API-ключ сохраняется в Windows Credential Manager

При последующих запусках:
- пользователь вводит PIN
- при успешной проверке открывается интерфейс чата
- при ошибке выводится сообщение

Также доступна кнопка **сброса ключа**, которая удаляет сохранённые данные и позволяет ввести новый ключ.

## Хранение данных

Используются два механизма хранения:

**SQLite**
- история сообщений
- статистика
- хэш PIN и salt

**Windows Credential Manager**
- API-ключ OpenRouter

Такой подход позволяет не хранить ключ в базе данных в открытом виде.

## Возможности

- работа с моделями OpenRouter
